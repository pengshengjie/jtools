# windows 好用的windows效率工具 Jtools

基于Tauri构建的程序员桌面效率工具

下载地址: https://raw.githubusercontent.com/pengshengjie/jtools/main/release/jtools.exe

🔥Feature:
  + 快速查找Chrome标签
  + 支持中英翻译
  + 自动打开VSCODE项目(开发中...)

🔥指令：
  + Ctrl+Space: 最小化(取消最小化)
  + g xxx: 搜索 官方文档xxx
  + fc xxx: 翻译 xxxx 成中文
  + fe: 翻译 xxxx 成英文
  + xxxx: 查找xxx软件 或者VScode项目xxx 或者浏览器标签xxx
